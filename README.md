# JudgeBot

Арбитражный бот, получающий данные об асках и бидах пар криптовалют по WS и HTTP запросам с целого ряда разилчных криптобирж.
Определяет спред и отправляет самые выгодные комбинации в телеграмм бот. 

Идея реализована до конца, но по факту оказалось, что большинство сильных перепадов в ценах между биржами объясняются проблемами с переводами конкретных валют или отсуствием поддержки определенных сетей на биржах. Увы =)
